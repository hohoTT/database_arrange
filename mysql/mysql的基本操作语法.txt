首先进入mysql：

mysql -u root -p password





------------------------------------------------------------------------------------------------------------------
有关mysql系统配置的相关操作：


1.修改mysql中用户的密码：

set password for root@localhost = password('新的密码'); 


2.查看数据库当前编码：

show variables like 'character%';


3.mysql设置编码命令:

SET character_set_client = utf8;

SET character_set_connection = utf8;

SET character_set_database = utf8;

SET character_set_results = utf8;

SET character_set_server = utf8;


注意：此种方式只在当前状态下有效，当重启数据库服务后失效。

所以如果想要不出现乱码只有修改my.ini文件，
从my.ini下手(标签下没有的添加，有的修改)

[client]

default-character-set=utf8

[mysql]

default-character-set=utf8

[mysqld]

default-character-set=utf8

------------------------------------------------------------------------------------------------------------------
基础的sql语句：

1.查看数据库：

show databases;


2.使用数据库：

use 数据库名


3.查看某个数据库中的表(这里要注意的是需要先使用某个数据库):

show tables;


4.查看数据库中的某一张表

select * from 表名;


5.以网格的形式查看数据库表（这样可以在表比较庞大的时候给我以清晰的数据结构）

select * from 表名\G;


6.删除数据库表:

drop table 表名;


7.删除数据库:

drop database 数据库名;


8.查看数据库表的结构：

desc 表名;




------------------------------------------------------------------------------------------------------------------
以下是有关表的相关操作：

1.查看创建表时的创建内容：

show create table 表名;


2..查看表的结构：

show columns from 表名;







------------------------------------------------------------------------------------------------------------------
运用聚合函数进行查询：

1.求表中某一列的平均值：

select avg(列名) from 表名;


2.求表中某一列的平均值（保留几位小数，例子中为保留两位小数）：

select round(avg(列名), 2) from 表名;


3.查询表中大于等于多少的数据：----------------这里还可以是其他的查询条件

select * from 表名 where 列名 >= 某一数字;
